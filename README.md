# BalanzIt – Budget Planner

BalanzIt is a budget planning web application that helps users track expenses, visualize spending, and generate detailed financial reports.

## Features

* **Expense Tracking** – Add and manage your daily expenses
* **Interactive Pie Chart** – Visualize spending by category using Recharts
* **Savings Goal** – Set and monitor your savings target
* **PDF Report Generation** – Export a full report including:
  * Expense table
  * Budget summary
  * Pie chart visualization
* **Real-time Updates** – Instantly see changes reflected across the application

---

## Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** CSS
* **Charts:** Recharts
* **PDF Generation:** jsPDF + html2canvas
* **Deployment:** GitHub Pages
